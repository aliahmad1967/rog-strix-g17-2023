\# ROG Strix G17 (2023) - Complete Website



A fully responsive, animated clone of the ASUS ROG Strix G17 2023 gaming laptop website.



\## Features



\- \*\*Custom Cursor\*\*: Dual-layer cursor with hover effects

\- \*\*Particle System\*\*: Interactive canvas animation in hero section

\- \*\*Scroll Animations\*\*: Reveal animations using Intersection Observer

\- \*\*Audio Visualizer\*\*: Animated canvas visualization

\- \*\*RGB Controller\*\*: Interactive keyboard color picker

\- \*\*Temperature Simulation\*\*: Live CPU/GPU temperature display

\- \*\*Testimonial Slider\*\*: Touch-enabled carousel

\- \*\*Design Gallery\*\*: Interactive image gallery

\- \*\*Responsive Design\*\*: Mobile-first approach



\## File Structure

│   ├── icons/             # SVG icons

│   └── testimonials/      # Avatar images

└── README.md





\## Setup



1\. Create the folder structure as shown above

2\. Place all CSS files in `css/` folder

3\. Place all JS files in `js/` folder

4\. Download and place images in appropriate `images/` subfolders

5\. Open `index.html` in a browser
